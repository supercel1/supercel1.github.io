熱伝導の微分方程式

フーリエが考え出した簡単な熱伝導の微分方程式からフーリエ級数を導いてみましょう

\[ \frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} \]

ここで、時間を\(t\)、ある長さ\(x\)の棒の温度を\(u(x, t)\)と置いています

この微分方程式を解いていきましょう。まずは、境界条件と初期条件を与えます。

境界条件

ある長さ$ $棒の両端が温度\(0\)である場合

\[ u(0, t) = u(\pi, t) = 0 \]

と表せます。

初期条件

\(t = 0\)の時の温度を\(f(x)\)で与えます

\[ u(x, 0) = f(x) \quad (0 \leq x \le \pi) \]

フーリエは熱伝導方程式の性質 - 解\(u_1\), \(u_2\)が存在するとき、\(u_1+u_2\)も解となる - 解\(u_1\)が存在するとき、定数倍\(ku\)も解となる

をみたすことを利用して解を求めることにしました。

  • まず境界条件を満たす簡単な形の解を求める
  • その後その解を足し合わせることで初期条件を満たす解を求める

では、解を求めていきましょう

まず、簡単な形の式とは変数分離型の式です

\[ u(x, t) = v(x)w(t) \]

として、式を解いてみましょう。

\[ \ddot{v} = Cv(x), \dot{w} = Cw(t) \]

まず時間の常微分方程式を解くと、\(w(t) = Ae^{Ct}\)と表されます。\(t \rightarrow \infty\)の時に\(w(t)\)が無限に大きくなることはないので\(C \leq 0\)であることがわかります。そこで、\(C = -n^2\)とおくと\(\ddot{v} = -n^2v\)となります。この常微分方程式の解は

\[ v = B\sin (nx + \theta) \]

となります。この解のうち境界条件を満たすのは、\(\theta = 0\)かつ\(n\)が整数であるときに限られます。よって、解は

\[ e^{-n^2t}\sin(nx) \quad (n = 1, 2, 3, \cdots) \]

となります（また、この定数倍）
