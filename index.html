熱伝導の微分方程式

フーリエが考え出した簡単な熱伝導の微分方程式からフーリエ級数を導いてみましょう

\[ \frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} \]

ここで、時間を\(t\)、ある長さ\(x\)の棒の温度を\(u(x, t)\)と置いています

この微分方程式を解いていきましょう。まずは、境界条件と初期条件を与えます。

境界条件

ある長さ$ $棒の両端が温度\(0\)である場合

\[ u(0, t) = u(\pi, t) = 0 \]

と表せます。

初期条件

\(t = 0\)の時の温度を\(f(x)\)で与えます

\[ u(x, 0) = f(x) \quad (0 \leq x \le \pi) \]

フーリエは熱伝導方程式の性質

  • 解\(u_1\), \(u_2\)が存在するとき、\(u_1+u_2\)も解となる
  • 解\(u_1\)が存在するとき、定数倍\(ku\)も解となる

をみたすことを利用して解を求めることにしました。

  • まず境界条件を満たす簡単な形の解を求める
  • その後その解を足し合わせることで初期条件を満たす解を求める

では、解を求めていきましょう

まず、簡単な形の式とは変数分離型の式です

\[ u(x, t) = v(x)w(t) \]

として、式を解いてみましょう。

\[ \ddot{v} = Cv(x), \dot{w} = Cw(t) \]

まず時間の常微分方程式を解くと、\(w(t) = Ae^{Ct}\)と表されます。\(t \rightarrow \infty\)の時に\(w(t)\)が無限に大きくなることはないので\(C \leq 0\)であることがわかります。そこで、\(C = -n^2\)とおくと\(\ddot{v} = -n^2v\)となります。この常微分方程式の解は

\[ v = B\sin (nx + \theta) \]

となります。この解のうち境界条件を満たすのは、\(\theta = 0\)かつ\(n\)が整数であるときに限られます。よって、解は

\[ e^{-n^2t}\sin(nx) \quad (n = 1, 2, 3, \cdots) \]

となります（また、この定数倍）

得られた簡単な解を足し合わせることで初期条件をみたす解を求めてみましょう。

\[ \sum_{n=1}^N b_n e^{-n^2t}\sin(nx) \]

まずは、\(N\)が有限である場合は上記の式になります。ここでフーリエは\(N \rightarrow \infty\)である場合、つまり

\[ u(x, t) = \sum_{n=1}^{\infty} b_n e^{-n^2t}\sin(nx) \]

も境界条件をみたすだろうと考えました。

初期条件をみたすには\(t = 0\)として、

\[ f(x) = \sum_{n=1}^{\infty} b_n \sin(nx) \]

となり、\(f(x)\)が三角関数の和として表されることになります。

有限和である場合に、両辺に\(\sin kx\)をかけて\(0\)から\(\pi\)まで積分すると、

\[ \int_0^{\pi} \sin nx \sin kx = \frac{\pi}{2} \quad (k = nのとき) \]

であるから、

\[ n\int_0^{\pi} f(x)\sin kx = \frac{n\pi}{2} b_n \]

\(n\)を消去して、

\[ b_n = \frac{2}{\pi}\int_0^{\pi}f(x)\sin kx \]

となります。

与えられた関数\(f(x)\)によって係数\(b_n\)が定まることがわかりました。得られた三角関数の和を\(f(x)\)のフーリエ級数と呼びます。

ベイズ学習計算練習

前提

  • 事前分布：ベータ分布
  • 事後分布：ベルヌーイ分布
  • \(\boldsymbol{x}\)：観測したデータ集合
  • \(\mu\)：\(\mu \in (0, 1)\)をみたすパラメータ

計算

ベイズ学習ではすべての変数は確率変数として計算していきます。そのため、上記のような前提を設けてパラメータを求めていきたいと思います。

目標：ベータ分布のパラメータ\(a, b\)をどのように更新するかを計算により求める

観測したデータ集合\(\boldsymbol{x}=(x_1, x_2, \cdots, x_n)\)があるとします。このとき、データ集合から求めようとする事後分布は\(p(\mu|\boldsymbol{x})\)となります。ベイズの定理より、

\[ p(\mu|\boldsymbol{x}) = \frac{p(\boldsymbol{x}|\mu)p(\mu)}{p(\boldsymbol{x})} \]

\(\mu\)からそれぞれのデータ\(\boldsymbol{x}\)が生成されてデータ間では無関係であるとみなすので、事後分布は

\[ p(\mu|\boldsymbol{x}) = \frac{\{\prod_{i=1}^n p(x_i|\mu)\}p(\mu)}{p(\boldsymbol{x})} \]

と表されます。さらに、一旦正規化を後回しにすることにして

\[ p(\mu|\boldsymbol{x}) \propto \{\prod_{i=1}^n p(x_i|\mu)\}p(\mu) \]

と表すことにします。

計算を簡単にするために対数をつけて計算することにします。

\[ \ln p(\mu|\boldsymbol{x}) = \sum_{i=1}^n \{x_i\ln\mu + (1 - x_i)\ln(1 - \mu) \} + \ln p(\mu) \]

\[ \ln p(\mu|\boldsymbol{x}) = \sum_{i=1}^n \{x_i\ln\mu + (1 - x_i)\ln(1 - \mu) \} + (a-1)\ln\mu + (b-1)\ln(1-\mu) + const \]

\(\mu\)でまとめると、

\[ \ln p(\mu|\boldsymbol{x}) = (\{\sum_{i=1}^n x_i\} + a-1)\ln \mu + (n - \{\sum_{i=1}^n x_i\}+b - 1)\ln(1-\mu) + const \]

となります。得られた事後分布はどこかベータ分布と似たような式になっていますよね？\(\hat{a}=a+\sum_{i=1}^n x_i\)、\(\hat{b}=n - \{\sum_{i=1}^n x_i\}+b\)とおけば、

\[ p(\mu|\boldsymbol{x}) = Beta(\mu|\hat{a}, \hat{b}) \]

と表せます。つまり、前提のように確率分布を定めると更新するべきパラメータの値を求めることができるようになるのです。このように事後分布と事前分布が等しくなるように設定された事前分布を共役事前分布と呼びます。

※解析的に計算できるのは特殊であり、計算できない場合は様々な近似法を用いることになります。

確率分布まとめ

ベルヌーイ分布

  • 離散確率分布
  • 二値をとる変数を生成する
  • \(x \in \{0, 1\}, \mu \in (0, 1)\)
  • 同時に起こらない二つの事象を扱うときに用いる

\[ Bern(x|\mu) = \mu^x (1 - \mu)^{1 -x} \]

カテゴリ分布

  • 離散確率分布
  • ベルヌーイ分布を多変数に拡張したもの
  • \(\boldsymbol{s}\)：\(K\)次元ベクトル、各要素は\(s_k \in \{0,1\}\)かつ\(\sum_{k=1}^Ks_k=1\)をみたす変数を生成してくれる
  • ex) \(K=4\)のとき、\(s_k=(0, 0, 1, 0)\)
  • \(\boldsymbol{\pi}=(\pi_1, \cdots, \pi_k)^T\), \(\pi_k \in (0, 1),\sum_{k=1}^K\pi_k=1\)をみたす

\[ Cat(\boldsymbol{s}|\boldsymbol{\pi}) = \prod_{k=1}^K\pi_k^{s_k} \]

ベータ分布

  • 連続確率分布
  • \(\mu \in (0, 1)\)となる変数を生成してくれる

\[ Beta(\mu|a, b) = C_B(a, b)\mu^{a-1}(1 -\mu)^{b-1} \]

  • \(a, b \in \mathbb{R}\)
  • \(C_B (a, b) = \frac{\Gamma{(a+b)}}{\Gamma{(a)} \Gamma{(b)}}\) 正規化するための項

ディリクレ分布

  • 連続確率分布
  • ベータ分布を多次元に拡張
  • \(\boldsymbol{\pi}=(\pi_1, \cdots, \pi_k)^T\), \(\pi_k \in (0, 1),\sum_{k=1}^K\pi_k=1\)をみたす変数を生成してくれる
  • \(C_D(\boldsymbol{\alpha})= \frac{\Gamma(\sum_{k=1}^K \alpha_k)}{\prod_{k=1}^K \Gamma(\alpha_k)}\)

\[ Dir(\boldsymbol{\pi}|\boldsymbol{\alpha})= C_D(\boldsymbol{\alpha})\prod_{k=1}^K \pi_k^{\alpha_k-1} \]

ガウス分布

  • 連続確率分布
  • \(x \in \mathbb{R}\)となる変数を生成してくれる
  • \(\mu \in \mathbb{R}\)：平均パラメータ
  • \(\sigma^2 \in \mathbb{R}\)：分散パラメータ

\[ \mathcal{N}(x|\mu,\sigma^2) = \frac{1}{\sqrt{2\pi \sigma^2}}exp\{-\frac{(x-\mu)^2}{2\sigma^2}\} \]

ポアソン分布

  • 連続確率分布
  • 非負の整数\(x\)を生成してくれる
  • \(\lambda \in \mathbb{R}\)

\[ Poi(x|\lambda) = \frac{\lambda^x}{x!}e^{-\lambda} \]

\[ \ln Poi(x|\lambda) = x\ln \lambda - \ln x! - \lambda \]
